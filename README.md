# Adding HTML Games to Protodactylus Game Hub

This guide will walk you through the steps required to add HTML games to the Protodactylus game hub.

## Prerequisites
- Ensure you have access to the repository: `stopcallingmegamersoup-wq/protodactylus`.
- Have your HTML game files ready for upload.

## Steps to Add a Game
1. **Prepare Your Game Files**: Ensure your HTML game files are organized in a single folder.

2. **Login to GitHub**: Access your GitHub account and navigate to the `stopcallingmegamersoup-wq/protodactylus` repository.

3. **Access the Code Section**: Click on the `<> Code` tab to view the repository's files.

4. **Create a New Folder for Your Game**: 
   - Click on the `Add file` button.
   - Choose `Create new file`.
   - In the text field for the filename, type the name of your new folder followed by a `/`. For example: `mygame/`.
   - Click on `Create a new file`.

5. **Upload Your Game Files**: 
   - In the new folder you created, click on the `Upload files` button.
   - Drag and drop your HTML game files or click on `choose your files` to select them from your device.

6. **Commit Your Changes**:
   - After uploading the files, scroll down to the `Commit changes` section.
   - Enter a commit message describing your changes (e.g., "Adding my HTML game").
   - Ensure the option `Commit directly to the main branch` is selected.
   - Click on `Commit changes`.

7. **Test Your Game**: 
   - Navigate back to the main page of the repository and locate your game’s folder.
   - Open the HTML file to test and ensure that the game runs properly.

## Additional Notes
- If you are adding multiple games, repeat steps 4 to 6 for each game.
- Ensure you follow the file structure conventions if there are specific required formats for the Protodactylus game hub.

## Conclusion
You should now have successfully added your HTML game to the Protodactylus game hub! Enjoy sharing your game with others!
